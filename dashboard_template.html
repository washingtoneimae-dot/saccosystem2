Setup Guide ▼

1. Gmail Setup (for sending emails)

  1. Create or use a Gmail account (e.g. norkesacco@gmail.com)
  2. Enable 2-Step Verification at Google Security
  3. Generate an App Password: Google Account → Security → App passwords → Select "Mail" + "Other" → name it "SACCO"
  4. Copy the 16-character password and update Himalaya config:
# ~/.config/himalaya/config.toml
[accounts.personal]
email = "your-sacco-email@gmail.com"
display-name = "Your SACCO Name"
default = true

backend.type = "imap"
backend.host = "imap.gmail.com"
backend.port = 993
backend.encryption.type = "tls"
backend.login = "your-sacco-email@gmail.com"
backend.auth.type = "password"
backend.auth.cmd = "echo YOUR_16_CHAR_APP_PASSWORD"

message.send.backend.type = "smtp"
message.send.backend.host = "smtp.gmail.com"
message.send.backend.port = 587
message.send.backend.encryption.type = "start-tls"
message.send.backend.login = "your-sacco-email@gmail.com"
message.send.backend.auth.type = "password"
message.send.backend.auth.cmd = "echo YOUR_16_CHAR_APP_PASSWORD"

2. Database — ./sacco.db (SQLite, no credentials needed)

3. API Server — Runs on port 9150. Start with: python3 ~/sacco/api_server.py

4. Dashboard — Open http://127.0.0.1:9150/dashboard in your browser

5. Customizing Sender Details — Edit the sender name/email in api_server.py (search for "From: Washington Imae") or set up the Settings feature to manage them from the dashboard.
